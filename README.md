# Planets.nu Plugin Suite

A suite of Tampermonkey userscripts for [Planets.nu](https://planets.nu) that add planetary management features to the game client.

## Plugins

### Economy Management (`economy-management.user.js`)
Adds a dashboard for managing planet builds and taxation across your empire. Supports custom build methods, tax strategies, and bulk automation across all your planets.

> **Testing version** — use at your own risk. Feedback welcome.

### Planet Info Overlay (`planet-info-overlay.user.js`)
Displays mineral, population, and resource overlays directly on the starmap.

> **Currently non-functional.**

## Installation

Install a userscript manager such as [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/), then open the `.user.js` file you want and click "Install".
